# 🎯 Guess My Number Game

A simple and interactive number guessing game built using **HTML, CSS, and JavaScript**.

---

## 🚀 Live Demo

👉 https://kanhujr.github.io/guess-my-number/

---

## 📌 Project Description

This is a browser-based game where the user tries to guess a randomly generated number between **1 and 20**.

* The game gives feedback like:

  * 📈 Too high
  * 📉 Too low
  * 🌈 Correct number
* Score decreases with each wrong guess
* Highscore is tracked during the session

---

## 🛠️ Tech Stack

* HTML5 → Structure
* CSS3 → Styling & Layout
* JavaScript (ES6) → Game Logic & DOM Manipulation
